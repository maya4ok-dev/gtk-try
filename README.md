# gtk-try

GTK examples and demos in C and C++

## what's this?

this is a project where I explore GTK and collect small examples and demos

it can also help others who are learning GTK or just want quick and working snippets without digging through docs

## contents

 * `hello-world.c` - creating an app and a window with a label
 * `let-window-be-window.c` - a test if you can store a window as a window, not a widget. this shows how GLib works
 * `app-hello-world.cpp` - using a plain C GTK in a C++ app, wrapping the app in a class
 * `clicker.c` - a simple clicker game that introduces signals and a box layout
 * `clicker-state.c` - the same clicker but using a state
 * `grid.c` - introduction to GTK grid layout
 * `log-viewer.cpp` - GTK frontend for my logging library
 * `drag-and-drop.c` - handling drag-and-drop events
 * `audio-player.c` - simple GStreamer-based audio player with drag-and-drop support
 * `todo.c` - a simple todo app with keyboard controls

## about comments

there are some comments in the code that explain how GTK works

they were written while I was learning GTK so they're really useful
